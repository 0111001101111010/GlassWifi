Glass Wifi 
=================
<b> Intro 
-----
Helps you connect to wifi at public spots

Install
---
adb install wifiplease.apk


use
---

by voice - ok, glass wifi please

navigate to glass

scroll to the card. "Wifi please". Enjoy
